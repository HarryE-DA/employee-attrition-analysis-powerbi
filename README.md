# Employee Attrition Analysis Dashboard



## Project Overview



This project analyses employee attrition using Power BI to identify factors contributing to employee turnover and provide data-driven recommendations to improve employee retention.



## Tools Used



- Power BI



- Microsoft Excel



- DAX



- Data Visualisation



## Dashboard Features



- Overall Attrition Rate



- Total Employees



- Employees Left



- Attrition Rate by Department



- Attrition Rate by Job Role



- Overtime vs Attrition Analysis



- Work-Life Balance Analysis



- Interactive Department and Overtime Slicers



## Key Findings



- Overall attrition rate: 16.1%



- Sales department recorded the highest attrition rate (20.6%)



- Sales Representatives recorded the highest attrition rate (39.8%)



- Employees working overtime experienced significantly higher attrition (30.5%)



- Employees with poor work-life balance showed increased attrition rates



## Recommendations



1. Improve workplace environment and employee satisfaction.



2. Review overtime practices and workload distribution.



3. Introduce retention initiatives for Sales Representatives.



4. Improve onboarding and development opportunities for employees with less than two years of service.
